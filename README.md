# 🤖 Pychatbot — Analyse de discours présidentiels

Projet Python de traitement du langage naturel (NLP) 
appliqué aux discours des présidents français.

## 📌 Description
Analyse d'un corpus de discours présidentiels via 
des techniques de NLP — pondération TF-IDF, 
extraction de mots-clés et similarité cosinus 
pour la sélection automatique de réponses.

## ⚙️ Fonctionnalités
- Calcul de la matrice TF-IDF sur le corpus
- Détection des mots les plus et moins importants
- Analyse par président (ex : mots les plus répétés)
- Identification des thèmes récurrents
- Interface conversationnelle (6 requêtes disponibles)

## 🛠️ Technologies
Python · TF-IDF · NLP · Similarité cosinus
