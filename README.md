FONCTIONNALITES PRINCIPALES

Analyse de Texte : Prétraitement et analyse de corpus de textes pour extraire des informations pertinentes.
Matrice TF-IDF : Création d'une matrice TF-IDF pour évaluer l'importance des mots dans le corpus.
Identification de Mots Clés : Détection des mots les plus fréquents et les moins importants dans le corpus.
Analyse Spécifique des Discours Présidentiels : Extraction de mots clés et de thèmes récurrents dans les discours des présidents français.

INSTRUCTIONS FONCTIONNALITES DE MY FIRST CHATBOT

Pour utiliser "My First Chat Bot", suivez ces étapes simples :

Une fois lancée, vous aurez le choix entre 6 fonctionnalités différentes. Vous pouvez sélectionner une fonctionnalité en tapant le numéro correspondant (de 1 à 6) :

1. Afficher la liste des mots les moins importants dans le corpus de documents. Un mot est considéré comme non important si son score TF-IDF est égal à 0 dans tous les fichiers.
2. Afficher le(s) mot(s) ayant le score TF-IDF le plus élevé.
3. Affihcer le(s) mot(s) le(s) plus répété(s) par le président Chirac.
4. Afficher le(s) nom(s) du (des) président(s) qui a (ont) parlé de la « Nation » et celui qui l’a répété le plus de fois.
5. Afficher le nom du premier président à parler du climat et/ou de l’écologie.
6. Affihcher le(s) mot(s) dits « non importants »,que tous les présidents ont évoqués.
Après avoir sélectionné une fonctionnalité, suivez les instructions à l'écran pour obtenir les résultats souhaités.
